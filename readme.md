# Boilerplate

Boilerplate is a bit overkill of a template, but covers a lot of my most used crates and features. Get coding faster.

## Included crates
 - serde: Serialize and Deserialize
   - serde_json:
 - yansi: Simple terminal text color implementation
 - std: Standard library includes
   - io::stdin: Standard input
   - fs: Work with files.
   - thread: Multithreading.
   - time::Duration: Sleep/delay
